keyboard-thumbstick.ino

- I wanted to free up 'wasd' and use keyboard directional keys for movement with a gamepad thumbstick.

- MakerHawk Arduino Pro Micro controller(DTH-11-L000244).  [Amazon](https://www.amazon.com/dp/B0D83FBYPD?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
- AKNES HallPi V6 TMR Joystick PS5.  [Amazon](https://www.amazon.com/dp/B0DNMCBR4N?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
